üöÄ API Endpoints

Comprehensive RESTful API with 12 endpoints supporting all category operations, organization filtering, and intelligent matching.

GET /api/v1/categories

Retrieve all categories with optional subcategories and statistics

GET /api/v1/categories/{id}

Get detailed category information with subcategories and top organizations

GET /api/v1/categories/{id}/subcategories

List subcategories for a specific category with statistics

GET /api/v1/categories/{id}/organizations

Find organizations by category with advanced filtering options

GET /api/v1/categories/{id}/recommended

Get recommended organizations for a category based on expertise

GET /api/v1/subcategories/{id}/organizations

Find organizations by subcategory with detailed capability filtering

GET /api/v1/capabilities/search

Search capabilities by keyword with category and proficiency filtering

POST /api/v1/matching/find

Find matching organizations using intelligent algorithm with scoring

PUT /api/v1/organizations/{id}/categories/{category_id}

Update organization category specialization (authenticated)

PUT /api/v1/organizations/{id}/subcategories/{subcategory_id}

Update organization subcategory specialization (authenticated)

GET /api/v1/categories/{id}/statistics

Get comprehensive category statistics and analytics

POST /api/v1/matching/update-scores

Update matching scores for all organizations (admin only)

üß† Intelligent Matching Algorithm

Advanced multi-factor scoring system that evaluates organizations across 6 key dimensions to provide the best matches for quote requests.

üéØ Category Expertise (30 points)

Evaluates organization's expertise level, years of experience, portfolio items, and certifications in the relevant category.

üí∞ Budget Compatibility (20 points)

Assesses alignment between quote budget and organization's typical project value ranges with flexibility scoring.

‚è∞ Timeline Compatibility (15 points)

Analyzes organization's typical delivery times against project deadlines with buffer time consideration.

üìç Location Proximity (10 points)

Considers geographic proximity or remote service capabilities for project delivery optimization.

‚≠ê Organization Quality (15 points)

Evaluates verification status, document completeness, team size, and years in business.

‚úÖ Requirements Matching (10 points)

Matches specific requirements like certifications, team size, languages, and technical skills.

üéØ Scoring System: Organizations receive a score out of 100 points. Matches above the configurable threshold (default: 50) are returned with detailed reasoning for each scoring factor.
